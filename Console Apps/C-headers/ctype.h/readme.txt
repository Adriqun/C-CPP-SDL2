Biblioteka ctype.h zawiera funkcje obs³ugi znaków 2016-03-07

isalnum - sprawdza czy znak jest litera albo cyfra ( 0, a, A )
isalpha - sprawdza czy znak jest litera ( a, A )
isblank - sprawdza czy znak jest bialy ( spacja, \n )
iscntrl - sprawdza czy wystapil znak nowej lini ( \n )
isdigit - sprawdza czy znak jest cyfra ( 0 )
isgraph - sprawdza czy znak jest graficzny tzn nie bialy ( a )
islower - sprawdza czy znak jest mala litera ( a, g )
isprint - sprawdza czy znak nadaje sie do druku ( a, 0, B )
ispunct - sprawdza czy znak jest punktem ( ! , . )
isspace - sprawdza czy znak jest bialy ( spacja )
isupper - sprawdza czy znak jest duza litera ( A, B )
isxdigit - sprawdza czy znak jest hexadecymalny ( 1, 7, f, a, E, B )

tolower - zmienia znak duzy na znak maly ( A -> a )
toupper - zmienia znak maly na znak duzy ( a -> A )
